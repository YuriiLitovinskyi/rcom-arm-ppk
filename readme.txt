Утиліта для відправки команди "Взяти групу під охорону" на ппк.
Запускається на хості з РКОМом.
РКОМ та Монго повинні бути запущені.

Для відправки команди необхідно ввести наступну інформацію:
 - номер ппк;
 - номер групи (1-4 для - 4L, та 1 - для 8L);
 - серійний номер приладу;
 - пароль доступу до ппк.

Команда буде відправлена при наступних умовах:
 - номер ппк існує в ркомі;
 - ппк в даний момент на зв'язку;
 - введена група існує в конфігурації ппк;
 - введена група у даний момент знята з охорони.

Команда буде виконана приладом при наступних умовах:
 - серійний номер приладу співпадає;
 - пароль доступу співпадає;
 - в конфігурації ппк (4L) надано доступ на управління групою через Інтернет;
 - шлейфи групи знаходяться в стані Норма.

При управлінні прилад відправить на пцс інформацію "Ідентифікація хозоргана 29"
